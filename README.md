## Always on Top (by Ankicraft)

You're studying in another app, referencing notes on the side, and every time you want to add a card, the Add Card window disappears behind everything else. 🗂️

Always on Top gives Anki the window pinning it's missing.

* 📌 **Pins the Add Card window above all other windows** — no more hunting for it.
* ⌨️ **Toggle on/off** with a fully customizable keyboard shortcut (default: **Ctrl+Shift+T**).
* 🔔 **On-screen tooltip** confirms the state every time you toggle.
* ⚙️ **One-field config dialog** — no clutter.
* 🌍 **UI translated** into 50+ languages.
* 🖥️ **Native Windows API (SetWindowPos)** for reliable topmost behavior; Qt flags + wmctrl fallback on macOS/Linux.

## How to Use

1. Open the Add Card window (press **A** or **Tools → Add**).
2. Press **Ctrl+Shift+T** to pin it on top — a tooltip confirms.
3. Press again to unpin.
4. Customize the shortcut any time via Config.

## Settings

Go to **Tools → Add-ons → Always on Top (by Ankicraft) → Config** to change the shortcut key.

## Compatibility

Tested on Anki 24.x (Windows). Should work on macOS and Linux via Qt/wmctrl fallback.
